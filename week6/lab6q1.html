JavaScript Pop-up Boxes Example

In JavaScript, you can create different types of pop-up boxes to interact with users:

1. Alert Box

Alert boxes display a message to the user.

2. Confirm Box

Confirm boxes ask for user confirmation with "OK" and "Cancel" buttons.

3. Prompt Box

Prompt boxes prompt the user for input.
